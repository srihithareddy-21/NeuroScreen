# Early Detection of Mental Health using Behavioural and MODMA Dataset

## Project Status
This project is currently under development.

## Project Overview
This project focuses on the early detection of mental health conditions using behavioural data and the MODMA dataset. The aim is to analyze speech signals and behavioural patterns to identify early indicators of mental health disorders.

## Objective
The objective of this project is to develop a machine learning approach that can analyze speech and behavioural data to support early detection of mental health issues.

## Dataset
The project uses the MODMA (Multimodal Open Dataset for Mental Disorder Analysis) dataset, which contains speech recordings and related behavioural data used for mental health research.
